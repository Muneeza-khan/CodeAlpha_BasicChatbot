# CodeAlpha_BasicChatbot
This project is a simple rule-based chatbot developed using Python as part of the CodeAlpha Python Programming Internship.
Features
- Responds to predefined user inputs
- Continuous conversation using loops
- Exit command support
- Beginner-friendly Python project
Technologies Used
- Python
Concepts Used
- Functions
- while loop
- if-elif-else
- Input / Output handling
Supported Commands
- hello
- how are you
- what is your name
- who created you
- bye
How to Run

```bash
python chatbot.py
```

Sample Output

```text
You: hello
Bot: Hi! How can I help you?

You: how are you
Bot: I'm fine, thanks for asking!

You: bye
Bot: Goodbye! Have a nice day.
```
